Hexagonal TodoMVC

What is hexagonal architecture?

Create your application to work without either a UI or a database so you can run automated regression-tests against the application, work when the database becomes unavailable, and link applications together without any user involvement.

Read more about it from Uncle Bob and Alistair Cockburn

The core business logic is bundled into a single javascript library local storage as the data source. You can see how we can easily swap out the storage to say Firebase or a custom REST api. Or switch to EmberJS or create a mobile version that doesn't lose the hard work of the main site.
